Readme for Autofac Developers

This document explains the developer setup and build execution for Autofac.

Developer Environment

  • Visual Studio 11, which includes:
    • .NET 4.0
    • ASP.NET MVC3
    • WCF RIA Services
  • NUnit Test Adapter for VS11 (to run unit tests inside Visual Studio)
  • SQL Server Express 2012 (for demo applications)
  • Sandcastle (for building documentation)

Building the Project

Developer build:
msbuild default.proj

Production/Release build:
msbuild default.proj /p:AssemblyVersion=1.2.3.4
